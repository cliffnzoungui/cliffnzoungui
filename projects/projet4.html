Application de gestion de contrats — Lesieur Cristal

Développement d'une application web permettant de centraliser, créer et suivre l'ensemble des contrats de l'entreprise. L'application intègre la gestion des échéances, le suivi des renouvellements, un système de droits d'accès par rôle et des notifications automatiques pour améliorer la traçabilité et réduire les risques d'oubli.

Outils & Technologies utilisés

⚡ JavaScript
🌐 HTML / CSS
🖥️ PHP
🗄️ MySQL
🔐 Gestion des rôles (Auth)
📧 Notifications email
🔍 Recherche avancée
📁 Archivage de documents
