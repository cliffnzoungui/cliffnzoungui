Infrastructure de Supervision de Serveurs Web

Mise en place d'une solution complète de monitoring pour superviser en temps réel l'état et les performances de serveurs web en entreprise. Déploiement d'un système d'alerting automatique permettant de détecter rapidement toute anomalie ou indisponibilité des services.

Outils & Technologies utilisés

🔍 Zabbix
📊 Grafana
🔥 Prometheus
🐧 Linux / Bash
📡 SNMP
🌐 Apache / Nginx
